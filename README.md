# 歌单墙 / Playlist Wall

歌单墙是一个面向网易云音乐的非官方桌面客户端。

它既能承担日常播放，也把分散、难翻、难搜的歌单，摊成一整面适合搜索、拖拽、归类和持续整理的“歌单墙”。

核心目标不是做另一个播放器，而是把“找歌”和“整理歌单”这件事做得更舒服。

如果你经常在“我喜欢的音乐”、自建歌单和收藏歌单之间来回搬运、重排和归档，它会比线性列表更顺手。

## v0.2.0 更新

- 新增“探索歌单”分区，应用打开后会静默预加载每日推荐与社区精选歌单
- 新增“艺人歌单”分区，可按艺人聚合曲库，并从右键菜单跳转到对应艺人歌单
- 支持探索歌单远程搜索、直接收藏探索歌单到“收藏订阅”
- 支持歌单内拖拽排序、跨歌单拖拽移动，拖动“我喜欢的音乐”时默认复制而不是移除源曲目
- 支持 `Ctrl` 多选、`Shift` 范围选择，并对选中歌曲做批量移除或批量加入歌单
- 支持删除订阅歌单后的 20 秒撤销提示
- 支持“我喜欢的音乐”显示模式、歌单折叠状态、推荐开关等偏好持久化
- 登录态优先使用 Electron `safeStorage` 保护；控制台写入异常时会自动降噪，避免 `EPIPE` 导致主进程崩溃

## 功能

- 高密度瀑布流一次展示全库歌单，按“自己创建 / 收藏订阅 / 探索歌单 / 艺人歌单”分区
- 启动后静默预加载探索歌单，切到探索分区时通常不需要再等首屏
- 探索分区支持每日推荐、社区精选和远程歌单搜索
- 可从探索歌单直接收藏到“收藏订阅”，并继续在墙上整理
- 可为任意歌单请求相似推荐曲目，一键加入目标歌单
- 右键支持移除歌曲、跳到艺人歌单、把歌曲转移到其他歌单
- 支持歌单内拖拽排序、跨歌单拖拽移动，以及“我喜欢的音乐”复制式拖拽
- 支持 `Ctrl` 多选、`Shift` 范围选中，对多首歌曲一起操作
- 可一键定位当前播放歌曲，自动切回正确分区并滚动到对应位置
- 支持主题切换、界面缩放、歌单折叠、“我喜欢的音乐”显示模式、推荐开关等偏好设置
- 二维码登录，不依赖本机客户端；歌单数据后台渐进加载并本地缓存

## 快速开始

```bash
npm install
npm start
```

首次启动后扫码登录，即可进入歌单墙。

## 测试与打包

```bash
npm run test:node
npm run test:ui
npm run dist:win
```

- `test:node`：主进程 `safe-console` 单元测试
- `test:ui`：Playwright UI 测试
- `dist:win`：生成 Windows 解压版和发布用 `.zip`

## 快捷操作

| 按键 | 功能 |
|------|------|
| `/` | 聚焦搜索框 |
| `Space` | 播放 / 暂停 |
| `1` / `2` / `3` / `4` | 切到“自己创建 / 收藏订阅 / 探索歌单 / 艺人歌单” |
| `P` / `N` | 上一首 / 下一首 |
| `Esc` | 关闭菜单、预览、撤销提示或设置面板，也可清空当前选择 |

## 注意事项

- 需要可用的网易云音乐账号
- 某些私密歌单或接口异常时，歌单可能暂时无法完整展开
- 某些歌曲受版权或账号权限影响，可能只有试听片段

## 本地数据与登录态

- 登录态、偏好设置、播放统计和歌单缓存会写入 Electron 的 `userData` 目录，不会回写到仓库目录
- 登录 cookie 保存在 `userData/session.json`
- 在支持的平台上，程序会优先使用 Electron `safeStorage` 对 cookie 做系统级保护后再落盘
- 如果当前环境不支持 `safeStorage`，程序仍会保持登录态，但会退回到明文 `session.json` 持久化，并在应用内给出警告
- 旧版本留下的明文 `session.json` 会在下次成功读取时自动迁移到受保护存储；如果当前环境不支持安全存储，则会继续按明文格式保留
- 在共享设备上使用时，建议定期退出登录，或手动清理 `userData` 目录中的本地缓存与登录态文件
